### Conversion Events (in App tracken)
```js
// Free-Tool genutzt
fbq('track', 'Lead', { content_name: 'DSGVO-Audit' });

// Professional Sign-up
fbq('track', 'CompleteRegistration', { content_name: 'Professional' });

// Stripe-Purchase (nach Webhook-Confirmation)
fbq('track', 'Purchase', { value: 149.00, currency: 'EUR' });
```

> **DSGVO-Hinweis:** Pixel **erst nach Consent** laden (Cookie-Banner Reject-Default).
> Sonst Bußgeld-Risiko (siehe LG München I Google-Fonts-Urteil).
