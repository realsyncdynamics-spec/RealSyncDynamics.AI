# 10_products

Per-product technical fact sheets. One document per product,
created from `../90_templates/product-template.md`.

Each fact sheet describes the product in operational terms:
purpose, boundaries, features, data handling, evidence emission,
dependencies, and operational SLOs. No marketing copy, no
roadmap commitments, no customer references.

Naming convention: `<product-slug>.md` (lowercase,
hyphen-separated).
