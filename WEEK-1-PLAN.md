## If You Get Stuck

| Problem | Quick Fix |
|---------|----------|
| Scanner hangs | Check network timeout (default 10s), increase if needed |
| PDF layout broken | Verify @react-pdf/renderer version, simplify template |
| Stripe webhook fails | Check env var, verify webhook URL registered in dashboard |
| Mobile UI broken | Use `max-width: 100%`, test on real device |
| Database errors | Run migrations: `supabase db reset` |
