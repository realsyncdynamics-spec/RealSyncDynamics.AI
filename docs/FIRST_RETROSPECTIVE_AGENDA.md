## Facilitator Tips

**Creating Psychological Safety:**
- Start with appreciation for the team's effort
- Frame improvements as system improvements, not personal criticism
- Celebrate wins explicitly
- Thank people for honest feedback

**Keeping Discussion Focused:**
- Use a timer: Don't let any topic run >5 min
- If discussion gets heated, table for 1:1 conversation
- Redirect tangents: "Let's capture this as a separate action item"
- Ask clarifying questions: "Can you give a specific example?"

**Ensuring Action:**
- Every item needs an owner before meeting ends
- Owners must confirm timeline is realistic
- Document everything in GitHub for visibility
- Review action items at start of next retrospective

**Handling Common Issues:**
- **Silence:** Ask direct questions to draw people out
- **Dominating speaker:** "Thank you, let's hear from others too"
- **Vague complaints:** "Can you be more specific? What exactly happened?"
- **Blame game:** "Let's focus on what we can improve, not who's at fault"
