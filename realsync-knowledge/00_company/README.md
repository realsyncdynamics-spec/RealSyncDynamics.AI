# 00_company

Company-level operational context. Documents in this folder
describe the organisation as it relates to platform operation:
ownership, registered locations, accountable roles, and high-
level operating context.

Out of scope here:
- Marketing positioning (not in this repository)
- HR / personnel records (not in this repository)
- Customer data (not in this repository)

New documents in this folder start from a template in
`../90_templates/`. There is no dedicated company template yet;
use `architecture-template.md` and adapt section headings as
needed.
